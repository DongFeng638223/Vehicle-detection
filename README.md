# Vehicle-detection
非法停放车辆自动检测警告软件：

1、文件说明
- combined_1.txt.model：分类器文件；
- interface.fig、interface.m：演示界面文件；
- Pedestrian.exe：车辆检测主程序；
- temp.txt：演示软件临时文件；
- test_video.mov：测试用视频文件；
- warning.wav：软件使用的警告声音音频文件。

2、用途 
- 在不允许机动车停放的路段，可以利用监控视频监控是否有非法停车发生；
- 记录下非法停车的证据，同时通过声音提醒非法停车的司机此处不可以停放机动车。

3、默认工作模式
- 默认在十秒内在监控区域内有车辆停靠，则进行持续警告，警告超过一定次数，则进行违法记录，使用图像记录违法行为；

4、可调参数
    目前软件没有将太多参数设为可调节，只做演示使用：
- 检测的灵敏度可根据场景自行调整。

4、软件使用
- 双击文件夹下的interface.fig文件，就可使用MATLAB打开演示界面；
- 点击“选择视频文件”按钮，选择想要检测的视频；
- 拖动“检测器灵敏度radio(典型值53)”滑动条，调节检测的灵敏度（数值越小灵敏度越高）；
- 点击“开始检测”，运行检测程序。
